# AQVEX Catalog — Frontend Test Task

This project is a technical assessment for the **AQVEX** company. It features a fully functional product catalog page built with a focus on performance, scalability, and strict adherence to the provided Figma design.

## 🚀 Tech Stack
* **React 19.2.0** with **Vite** — for a lightning-fast development environment.
* **TypeScript** — for robust data typing and improved developer experience.
* **Tailwind CSS** — for pixel-perfect, responsive UI implementation.
* **Custom Typography** — integrated Klein font family (ExtraLight to Bold).

## ✨ Key Features

- **Dynamic Data Fetching:** Seamless integration with the remote REST API endpoint.
- **Advanced Search:** Real-time product filtering by name.
- **Smart Sorting:** Sort products by popularity, newest arrivals, and price (low-to-high / high-to-low).
- **Pagination:** Smooth navigation through the product listing.
- **Modern UX/UI:**
  - Responsive layout for mobile, tablet, and desktop.
  - Custom UI elements: unique discount badges, tailored dropdowns, and interactive buttons.
  - **Skeleton Screens:** Elegant loading states to enhance perceived performance.
- **Robust Error Handling:** - "No results found" states for empty searches.
  - Fallback mechanisms for broken or missing product images.
